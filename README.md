## Dev info

### Start a development server (sveltekit)

`npm install` then `npm run dev`

### Build a production version

`npm run build`

Preview with `npm run preview`

### Index users

Navigate to /admin

### Env vars

- `SUPABASE_ANON_KEY` (See Community Archive docs)
- `OPENAI_API_KEY`
- `ADMIN_PASSWORD` (for indexing users)
- `DEBUG` (optional)
